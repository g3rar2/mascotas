# Mascotas

Aplicacion web de ejemplo para registro e inicio de sesion de usuarios, construida con Express, Pug, Sequelize y MySQL.

## Tecnologias

- Node.js
- Express
- Pug
- Sequelize
- MySQL
- Tailwind CSS

## Configuracion

1. Instalar dependencias:

```bash
npm install
```

2. Crear un archivo `.env` a partir de `.env.example`.

3. Ejecutar la aplicacion:

```bash
npm run server
```

4. En otra terminal, si se modifican estilos:

```bash
npm run css
```

## Seguridad

El archivo `.env` contiene datos locales de base de datos y no debe publicarse. Para compartir el proyecto, usar solamente `.env.example`.
